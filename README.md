# 🧠 Quiz Application (Java Swing GUI)

A graphical Quiz Application developed using Java Swing that allows users to answer multiple-choice questions through an interactive interface and receive instant results.

---

## 📌 Features

- Graphical User Interface using Java Swing
- Multiple-choice quiz system
- Next/Previous question navigation (if implemented)
- Instant score calculation
- Result display at the end of quiz
- Simple and interactive UI design

---

## 🛠️ Technologies Used

- Java
- Swing (GUI)
- AWT (Event Handling)
- Object-Oriented Programming (OOP)

---

## ▶️ How to Run

1. Open project in Eclipse / IntelliJ IDEA
2. Run the main class file (e.g., `QuizApp.java`)
3. Start the quiz from the GUI window
4. Select answers and proceed through questions
5. View final score at the end

---

## 🎯 Learning Outcomes

- GUI development using Java Swing
- Event-driven programming concepts
- Working with buttons, labels, and panels
- Logic building for quiz systems
- Score calculation and result handling

---

## 👨‍💻 Project Type

Graphical desktop application developed as a Java mini project using Swing.
